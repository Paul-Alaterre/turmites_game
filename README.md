# turmites_game
Projet en programmation orienté objet implémentant un jeu reposant sur le principe des Turmites.
Réalisé en mars 2025 dans le cadre du projet de fin de semestre de la matière "Programmation orienté objet en Python" en L1 d'Informatique.

Le but du jeu est de pouvoir simuler le comportement d'une ou plusieurs Turmites tout en choisissant des paramètres que sont le nombre de lignes et de colonnes et le nombre de couleur différent.

Pour lancer le jeu, éxecuter le fichier myApp.py.
